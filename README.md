# Super Smash Heroes

####  Arcade style fighting game with Super Heroes August 27, 2018

#### By **Connor McCarthy, Chris Crow, David Zhu, & Brian Nelson**

## Description

An arcade style fighting game that uses Marvel super heroes as it's characters.
